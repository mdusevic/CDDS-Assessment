Introduction
------------------

 A simple program that allows for the editing and displaying of small objects that move across the window.

------------------
Requirements
------------------

 To run the program you must have:

 - Windows 10

------------------
Installation
------------------

 Follow the steps below to access the binary tree

  1) Open .zip file

  2) Extract to desired folder location

  3) Open release folder 
  
  4) Run the CDDS_IPC_EntityEditor.exe file

 Optional:

  5) Run the CDDS_IPC_EntityDisplay.exe file. (Requires the editor file to be open)

------------------
Usage
------------------

    *How-To-Use*
    ------------
     While using the editor adjust the given tabs to create a new object to move across the screen. To
     release the object simply create a new object with arrow at the top of the editor.

     To view all the objects without the different tabs from the editor, open the display program.
     
     To exit simply press the 'ESC' button on your keyboard or close the program window.

     *WARNING* Terminating the console will lose progress made in the session. Program will also not run if
               installation steps are not properly followed.
