Introduction
------------------

 A simple phone directory that utilises hashing. It can insert, remove, search and display records in the
 directory.

------------------
Requirements
------------------

 To run the program you must have:

 - Windows 10

------------------
Installation
------------------

 Follow the steps below to access the double linked list

  1) Open .zip file

  2) Extract to desired folder location

  3) Open release folder 
  
  4) Run the Hashing.exe file

------------------
Usage
------------------

    *How-To-Use*
    ------------
     Once the program is running, simply follow the prompts on the screen to do various 
     things to the linked list.
     
     To access the display feature, enter the associated number in the main menu. A list of name with their
     associated phone numbers will appear. 

     To access the search feature, enter the associated number in the main menu. There you will be prompted
     to enter the associated phone number of the record you want to find. Once entered the record will be
     searched for and if found displayed.

     To access the create feature, enter the associated number in the main menu. There you will be asked to
     fill out the details of the new record. Do NOT exit the program during the process or data entered will
     be lost.
     
     To access the remove feature, enter the associated number in the main menu. There you will be asked to
     fill out the details of the existing record. Once entered the record will be searched for and if found
     deleted.

     To exit simply follow the exit prompts in the menus and enter the required input.
     This will end the program.

     *WARNING* Terminating the console will lose progress made in the session.
